Browser-based pixel art editing tool developed specifically for my custom javascript game engine (see js-game project), which allows me to specify per-pixel opacity, and exact rendering settings, such as anti-aliasing, by producing Uint8ClampedArray data that can be dynamically configured by Javascript for maximum control of image processing and rendering typically unattainable by existing png solutions.

Toolsets and features are exposed on the webpage itself. This tool can be used at the following link:
https://murphyvjamese-sudo.github.io/webApp-pixelArtEditor/
